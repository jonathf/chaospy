Welcome to chaospy's documentation!
===================================

Chaospy is a numerical tool for performing uncertainty quantification using
polynomial chaos expansions and Monte Carlo methods.

A article in Elsevier Journal of Computational Science has been published
introducing the software:
`here <http://dx.doi.org/10.1016/j.jocs.2015.08.008>`_.
If you are using this software in work that will be published, please cite this
paper.
